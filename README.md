# Персональний менеджер фінансів (ПМФ)

Вебдодаток для обліку та аналізу особистих доходів і витрат, розроблений у межах курсу **«Розробка вебдодатків на Ruby від SoftServe»**. Реалізовано CRUD для сутностей Категорія та Операція, валідації моделей, асоціації ActiveRecord, модуль генерації звітів (аналітика по категоріях і датах з використанням Chart.js), адаптивний інтерфейс на Bootstrap, модульні тести (RSpec) та деплой на віддалений сервер. Проєкт демонструє практичні навички роботи з Ruby on Rails, REST-архітектурою, базами даних і побудовою повноцінного MVC-застосунку.
